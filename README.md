# 🎓 Student Marks Calculator

A simple and interactive web application to calculate the **total** and **average marks** of a student using **HTML, CSS, and JavaScript**.

---

## ✨ Features

- 📥 Enter marks for three subjects  
- 🧮 Automatically calculates total marks  
- 📊 Displays average marks (with 2 decimal precision)  
- 🎨 Clean and simple user interface  
- ⚡ Instant results on button click  

---

## 🗂️ Project Structure

project-folder/  
├── index.html  
├── style.css  
└── script.js  

---

## 🚀 How to Use

1. Open the project folder  
2. Double-click **index.html**  
3. Enter marks in all three fields  
4. Click **Calculate**  
5. View your results instantly  

---

## 💡 How It Works

- The user inputs marks into three fields  
- JavaScript reads the values  
- Calculates:
  - ✅ Total = Sum of marks  
  - ✅ Average = Total ÷ 3  
- Results are displayed dynamically on the page
